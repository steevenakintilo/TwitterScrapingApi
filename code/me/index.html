def get_list_of_homepage_tweet(selenium_session,nb_of_tweet_to_search=25):

  • Explanation

    • This function will get a list of information about my last x homepage tweet

  • Parameters:

    • selenium_session -selenium browser
    • nb_of_tweet_to_search -number of tweet to retrieve (the maximum number of tweet you can search is up to 100 to avoid tweet search limitations)

  • Return:

    • If it succed a list of dictionaries for each tweet retrieved each one have the username of the tweet ,the text of the tweet, the id of the tweet, the url of the tweet, the date of the tweet, the number of like of the tweet, the number of retweet of the tweet and the number of reply
    • If it failed it return an empty list
